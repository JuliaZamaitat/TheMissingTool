# TheMissingTool

start docker container:

`docker-compose up`

stop docker container:

`docker-compose down`

run with nodemon:

`npm start`

seed the Database with:

`npm run db:seed`

open tests with cypress:

`npm run cy:open`

run tests with cypress on console:


`npm start && npm run cy:run`
